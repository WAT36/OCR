# OCR

---

```
# 入力画像を左右半分に分ける
python3 trimmer.py (入力画像のパス)

# 入力画像を文字起こしする
./tesseract.sh (入力画像のパス) (出力ファイル名(拡張子不要))

# 文字起こしデータをcsvに変換する
python3 csv_adapter.py (入力左半分の画像のパス) (入力右半分の画像のパス) (出力CSVファイル)

# ↑答えだけは読み取れないので　ここだけは後で手動で登録

答えの記号を手動で登録後に以下実行
python3 csv_finalizer.py (入力CSVファイル) (出力CSVファイル)
```
